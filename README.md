## Technologies

Project is created with:

- Node version: 16.14.0
- NPM version: 8.3.1
- FE - React - Typescript
- BE - NodeJs - Express - GraphQL

## Setup

```
# Install server dependencies
npm install

# Install client dependencies
cd client
npm install

# Run both Express & React from root
cd ..
npm run dev

# Build for production
cd client
npm run build

# To Test
cd client
npm run test
```
