## General info

## Technologies
Project is created with:
* Node version: 16.2.0
* NPM version: 7.14.0
* FE - React - Typescript
* BE - NodeJs - Express - GraphQL


## Setup

```

```bash
# Install server dependencies
npm install

# Install client dependencies
cd client
npm install

# Run both Express & React from root
npm run dev

# Build for production
cd client
npm run build
```
